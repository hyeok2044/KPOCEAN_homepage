Features of KPOCEAN CMS

Accessible

• Website is accessible from anywhere, any device with internet access.
• Real-time vessel tracking is supported through Marine Traffic.
• Connection with crew is possible with Facebook link
• Real-time Statistics can be viewed.

Quick

• Used Apache Web Server + PHP + MySQL + Javascript to handle the data.
• Optimized code and database can support up to 100,000 crews under 1 second.

Simple

• Page designs are simple enough for anyone to understand the page functions
• Sorting, searching and exporting the crew details is easy.
• Managing vessels, clients, and principals, and Accounting, generating crew exchange plans can be done with one-click.

Secure

• General vulnerabilities of the programs are tested and blocked. (SQL injections, XSS, RFI, CSRF)
• User ID’s and user permissions can only be generated and managed by the admin.
• Permissions can be depended on user ID’s.
• Data backups are performed once a day, and certificates can be constantly updated with cloud storage.
